# Loan Calculator

A JavaFX loan calculator that computes monthly payments and also includes a bouncing ball animation and a live clock.

## Features
- Calculates monthly loan payment
- JavaFX graphical interface
- Bouncing ball animation
- Live clock display


## How to Run
1. Open the project in IntelliJ or Eclipse
2. Make sure JavaFX is added to the project
3. Run `LoanCalculator.java`
